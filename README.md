# Portfolio and Blog Documentation

## Structure

This repository contains a portfolio and blog that is structured as follows:

- `index.html`: The main entry point of the website.
- `portfolio/`: A directory containing portfolio items, each in a separate HTML file.
- `blog/`: A directory containing blog posts, each in a separate Markdown file.
- `assets/`: A directory for images and other static assets.
